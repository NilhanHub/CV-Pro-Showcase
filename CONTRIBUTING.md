# Contributing

CV Pro is a commercial product and this repository is a source-free public showcase. Product source contributions cannot be accepted here.

Constructive product feedback, accessibility observations, and reproducible issues with this showcase are welcome through GitHub Issues or Discussions. Please describe the observed behavior, the browser or device used, and a clear expected outcome. Do not include CVs, customer information, credentials, payment details, or other private data.

Security reports must follow [SECURITY.md](SECURITY.md) rather than a public issue.
